## Task 1
Ping a website.

Command:
ping google.com

Observe:
- IP address
- Response time

---

## Task 2
Perform DNS lookup.

Command:
nslookup github.com

Observe:
- Server
- IP address

---

## Task 3
Use dig command.

Command:
dig openai.com

Observe:
- ANSWER SECTION
- Query time
- DNS records

---

## Task 4
Check MX records.

Command:
dig gmail.com MX

---

## Task 5
Check name servers.

Command:
dig google.com NS
