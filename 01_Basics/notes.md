# 🌐 Day 1 - Networking Basics

## 🧠 What is a Network?

A network is a group of devices connected together to share data.

### Examples:
- Mobile connected to WiFi
- Laptop connected to printer
- Computers in a lab

---

## 🌍 What is the Internet?

The internet is a network of networks.

It connects:
- Home networks
- Office networks
- Servers around the world

---

## 🆔 What is an IP Address?

An IP address is a unique identifier given to each device in a network.

### Example:
192.168.1.1

It works like a home address, but for devices.

---

## 🔁 Basic Communication Flow

1. User requests a website
2. Request goes to server
3. Server sends response
4. User sees output

---

## 🧩 Key Terms

- Device → Any connected system (phone, laptop)
- Router → Connects networks
- ISP → Internet Service Provider (Jio, Airtel)
- Server → Provides data
- Client → Requests data

---

## 🎯 Summary

- Network = connected devices
- Internet = network of networks
- IP address = identity of a device
