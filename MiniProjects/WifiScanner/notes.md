# WiFi Scanner

A simple Python networking project that scans and displays nearby WiFi networks using built-in Windows networking commands.

## Features

- Detects nearby WiFi networks
- Displays SSIDs
- Uses built-in system tools
- Beginner-friendly networking project

## Technologies

- Python
- Networking Basics
- Windows WLAN Commands

## Run

```bash
python wifi_scanner.py
```
