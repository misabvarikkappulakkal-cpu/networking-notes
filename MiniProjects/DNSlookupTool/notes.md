# DNS Lookup Tool

A simple Python networking project that performs DNS lookups for domain names.

## Features

* Get IP address of a domain
* Handle invalid domains gracefully
* Beginner-friendly networking project
* Uses Python's built-in socket library

## Requirements

* Python 3.x

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python dns_lookup.py
```

Enter a domain name when prompted:

```text
Enter a domain name: google.com
IP Address: 142.250.193.78
```

## Concepts Covered

* DNS (Domain Name System)
* Domain Names
* IP Addresses
* Socket Programming
* Exception Handling
