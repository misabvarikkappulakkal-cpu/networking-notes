# Network Traffic Monitor Notes

## What is Network Monitoring?

Network monitoring is the process of observing network traffic to detect:

* Performance issues
* Security threats
* Connectivity problems

---

## Why Monitor Traffic?

Benefits:

* Detect attacks
* Troubleshoot networks
* Analyze bandwidth usage
* Improve performance

---

## Important Protocols

### TCP

Transmission Control Protocol

Characteristics:

* Reliable
* Connection-oriented
* Ordered delivery

Examples:

* HTTP
* HTTPS
* SSH

---

### UDP

User Datagram Protocol

Characteristics:

* Fast
* Connectionless
* No delivery guarantee

Examples:

* DNS
* VoIP
* Streaming

---

### ICMP

Internet Control Message Protocol

Used for:

* Ping
* Error reporting
* Network diagnostics

---

### ARP

Address Resolution Protocol

Purpose:

Maps:

IP Address
↓
MAC Address

---

## Packet Sniffing

Packet sniffing is the process of capturing packets traveling through a network.

Common tools:

* Wireshark
* tcpdump
* Scapy

---

## Key Concepts

### Source IP

Sender address.

### Destination IP

Receiver address.

### Source Port

Application sender port.

### Destination Port

Application receiver port.

### Protocol

Communication method used.
