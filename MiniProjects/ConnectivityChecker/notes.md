## Ping

Tests connectivity to another host.

Example:

ping google.com

--------------------------------

## DNS Resolution

Converts:

google.com

to

142.250.x.x

--------------------------------

## socket Module

Used for networking operations.

Example:

socket.gethostbyname()

--------------------------------

## subprocess Module

Runs system commands.

Example:

subprocess.run()

--------------------------------

## Return Codes

0 = Success

Non-zero = Failure
